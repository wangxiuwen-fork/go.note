# 2. 开发环境

Go代码必须放在工作空间内。它其实就是一个目录，其中包含三个子目录：src 、pkg 、bin 

|        |                                                              |                                                       |
| ------ | ------------------------------------------------------------ | ----------------------------------------------------- |
| src    | 目录包含Go的源文件，它们被组织成包（每个目录都对应一个包）， |                                                       |
| pkg    | 目录包含包对象，                                             |                                                       |
| bin    | 目录包含可执行命令。                                         |                                                       |
| GOPATH | 环境变量                                                     | http://www.flysnow.org/2017/01/05/install-golang.html |
|        |                                                              |                                                       |
