5.4. 错误    

内置的error是接口类型    

error类型可能是nil或者non-nil。 nil意味着函数运行成功， non-nil表示失 败。     

5.4.1. 错误处理策略    

fmt.Errorf函数使用fmt.Sprintf格式化错误信息并返回。     

5.4.2. 文件结尾错误（ EOF）    

```go
in := bufio.NewReader(os.Stdin)
for {
	r, _, err := in.ReadRune()
	if err == io.EOF {
		break // finished reading
	} 
	if err != nil {
		return fmt.Errorf("read failed:%v", err)
	} 
	// ...use r…
}
```
